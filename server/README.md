## running the project

make sure you have the latest version of npm installed by running `npm i -g npm`

copy the `.env.example` file to a file named `.env` and populate the correct values for your project

1. Open the command line interface (CLI)
2. Navigate to the project directory
3. Run the command "npm install" to install the necessary dependencies
4. Run the command "node index.js" to start the project

## how to add collection to postman

1. Open Postman
2. Click on the "Import" button in the top left corner of the application
3. Select the "Import From Link" option
4. Paste the JSON link in the provided field
5. Click on the "Import" button
